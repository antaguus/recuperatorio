## Declaración de uso de IA

Se utilizó ChatGPT como apoyo durante el desarrollo de los codigos html y css.

Se utilizó para revisar y corregir HTML y CSS, especialmente en estructura semántica, layout, centrado de elementos, navegación y uso de flexbox.

La IA ayudó a detectar errores en márgenes, padding y organización del contenido, que fueron corregidos en el código.

Su uso facilitó la resolución de problemas técnicos y la comprensión de conceptos, aunque las decisiones finales de diseño fueron propias.
